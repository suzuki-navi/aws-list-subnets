# aws-list-subnets

command line tool to list AWS subnets.

## Usage

    $ aws_list_subnets
    $ aws_list_subnets --profile AWS_PROFILE_NAME
    $ aws_list_subnets --help

## Installation

    $ pip install aws-list-subnets
